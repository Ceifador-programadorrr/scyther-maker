# introdução #
<p> Scyther Maker é um software baseado no metasploit-framework e utiliza muitas das funcionalidades ja existentes nele.</p>
<p>para automação de payload de msfvenom/msfconsole</p>
<p> com alvo em celulares android. </p>
<p> objetivo: tornar o teste de segurança a dispositivos moveis mais acessivel e prático.</p>
<p>principal funcionalidade:</p>
<p>a troca automatica da versão sdk para a correta.</p>
 
 #  instalação e configuração #

- execute o setup para instalar as dependencias:
`bash setup.bash`

- inicie o scyther maker[sm.bash]
`bash sm.bash `

- alternativa silenciosa(Quiet Mode):
`bash sm_iniciar.bash`
